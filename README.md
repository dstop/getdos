多线程 http get 发包工具

编译: gcc get.c -pthread -o getdos

使用方法: 

$ ./getdos "url" thread

$ ./getdos "网址" 线程

例子:

$ ./getdos "http://www.xxx.com/new.php?id=5&p=5" 10

网址里面如果含&必须加双引号，线程可不填，默认线程5
